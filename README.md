# React-Events-Forms
## Getting Started

cd react-app
#
Install semantic-ui:
npm install semantic-ui-react semantic-ui-css
#
Start json server:
npm install -g json-server

Then:
json-server --watch db.json

#
Start your react app:
npm install

Then:
npm start
#
## Deliverables
